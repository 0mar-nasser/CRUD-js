# 🛠️ CRUD App | Product Management

A simple and responsive **CRUD (Create, Read, Update, Delete)** application built with **HTML, CSS, and JavaScript**.  
This project allows users to manage products easily — add, update, delete, and search by title or category.

---

## 🚀 Features
✅ Create new products with title, price, taxes, ads, discount, and category.  
✅ Update or delete existing products.  
✅ Search by **title** or **category**.  
✅ Calculate total price automatically.  
✅ Responsive and modern UI design.  
✅ Popup notifications for user actions.  

---

## 🧩 Technologies Used
- **HTML5** – Structure  
- **CSS3** – Styling and Layout  
- **JavaScript (ES6)** – Functionality and Logic
